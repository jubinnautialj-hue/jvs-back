### 加解密

文件目录：

`cn.bctools.gateway.utils.DataUtil`

目前使用AES进行加解密，确保加解密方式参数一致,需要与前端解密保持一致；使用其他加解密方式参考具体方式对应的工具类

前端加密方式请参数<前端解密加解密方式>

#### 建议只修改 `cn.bctools.gateway.utils.DataUtil#encodeBody` 此方法,不修改`cn.bctools.common.utils.PasswordUtil#encodePassword(java.lang.String, java.lang.String)`公共方法

#### 加密函数

```
  AES aes = getKey(appId);
  return aes.encryptHex(str.trim(), StandardCharsets.UTF_8);
```


#### 部分接口不需要处理加密时可能在平台加密忽略中界面配置

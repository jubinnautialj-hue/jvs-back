### API说明 

- 终端服务API `ApplyServiceApi`    
- 用户部门 api `AuthDeptServiceApi`    
- 岗位 api `AuthJobServiceApi`    
- 用户角色 api `AuthRoleServiceApi`   
- 租户级系统配置 api `AuthTenantConfigService`
- 租户 api `AuthTenantServiceApi`     
- 用户 api `AuthUserServiceApi`    
- 字典 api `DictApi`    
- 环境变量 api `EnvironmentVariableApi`   
- 文件保存 `FileApi`    
- 用户选择时搜索 api `SelectedServiceApi`
- 用户扩展信息查询 `UserExtensionServiceApi`  
- 用户分组或团队 api `UserGroupServiceApi`

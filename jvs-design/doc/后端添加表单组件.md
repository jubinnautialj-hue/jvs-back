
## 后端添加组件

### 添加组件声明

在`DataFieldType`类中添加组件声明`tableForm("表格", FormComponentType.container, false, Map.class, true, false),`


~~~
@Getter
public enum DataFieldType {

   ******
   tableForm("表格", FormComponentType.container, false, Map.class, true, false);
   ******
   
    @ApiModelProperty("描述")
    private final String desc;
    @ApiModelProperty("是否支持移动端")
    private final Boolean mobile;
    @ApiModelProperty("组件类别,容器组件,基础组件,高级组件,扩展组件")
    private final FormComponentType itemType;
    @ApiModelProperty("是否支持导出")
    private final boolean enableExport;
    @ApiModelProperty("Java类型")
    private final Class<?> aClass;
    @ApiModelProperty("是否支持数据联动")
    private final boolean enableDataLinkage;

}

~~~

### 根据组件定义数据结构

组件对象,需要继承`FieldBasicsHtml`实体类对象

~~~

@Data
@Accessors(chain = true)
@ApiModel(value = "表格")
@EqualsAndHashCode(callSuper = true)
public class TableFormItemHtml extends FieldBasicsHtml {

    @ApiModelProperty(value = "表格字段")
    private List<FieldPublicHtml> tableColumn;
    @ApiModelProperty(value = "模型关联的id")
    String dataModelId;
	
}
~~~

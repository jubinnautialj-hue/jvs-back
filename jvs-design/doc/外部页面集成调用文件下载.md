### * 下载文件

轻应用中下载文件只能通过逻辑引擎获取一个文件地址后，打开对应的下载链接地址进行下载

注： 操作模型只能通过逻辑引擎，只需要以文件类格式的节点返回即可


格式示例：

```json
{
   //桶名
    "bucketName":"jvs-public",
    //文件名
    "fileName":"目录.xlsx",
    //文件类型
    "fileType":".xlsx",
    //目录地址
    "module":"rule/run/excel",
    //源文件名
    "originalName":"目录.xlsx",
    //打开类型，前端根据此类型确定是预览还是打开
    "outputType":"download",
    //访问url地址
    "url":"http://ip:port/jvs-public/jvs-design-mgr/1/rule/run/excel/2023/01/15/2024-01-15931554330446565376-%E7%9B%AE%E5%BD%95.xlsx"
}
```
